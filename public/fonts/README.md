# Tan Pearl Font Setup

To use the "Tan Pearl" font for invitation guest names, please add the font file here.

## Instructions:

1. **Download the Tan Pearl font file** (.ttf or .otf format)
   - The font is available from various font websites
   - Make sure you have the proper license to use it

2. **Place the font file in this directory** (`public/fonts/`) with one of these names:
   - `TanPearl.ttf`
   - `TanPearl.otf`
   - `tan-pearl.ttf`
   - `tan-pearl.otf`
   - `Tan Pearl.ttf`
   - `Tan Pearl.otf`

3. **The code will automatically detect and use the font** when generating invitations

## Note:
If the font file is not found, the system will fall back to Times Italic or other serif fonts.
